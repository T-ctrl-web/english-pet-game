# 🎮 萌宠英语乐园

一款面向考研/四六级水平英语学习者的 **中英互译拼写游戏**，以萌宠培育为驱动力，融合游戏化学习体验。

> 🖥️ **Web 浏览器版** - 原生 HTML + CSS + JavaScript，无需安装，打开即玩

## ✨ 功能特色

### 🎯 核心玩法
- **中英互译拼写**：中文→英文 / 英文→中文 双向拼写输入
- **三级难度**：四级（EASY）/ 六级（MEDIUM）/ 考研（HARD）随机混合
- **无固定答案**：基础版使用本地题库精确匹配，后续可接入AI判定
- **语法纠错**：答完后展示详细的纠错与改进建议
- **连对奖励**：连续答对3题奖励1条命（上限10条）

### 🐾 萌宠培育
- **5种萌宠可选**：猫、狗、豚豚、噜噜、线条小狗
- **状态养成**：心情值 + 饱腹度双维度
- **积分兑换**：答题获得积分，在商城兑换食品/玩具
- **宠物互动**：投喂、玩耍，提升宠物状态

### 🎨 视觉风格
- 天蓝/淡蓝/青绿为主的**清新青春风**
- 圆润卡片设计，温暖治愈
- 丰富的动画与反馈效果

## 🔐 API Key 安全提醒

> ⚠️ **重要：请务必阅读**

本项目支持 AI 语义评测和 AI 实时出题功能，需要配置 API Key 才能使用。

### 安全须知
1. **不要在代码中硬写入 API Key！** API Key 请在页面中通过设置界面（⚙️齿轮图标）输入
2. API Key 会保存在浏览器 `localStorage` 中，**仅在你自己的浏览器本地存储**
3. 如果你 fork 或部署此项目到公开仓库，请确保 `js/aiService.js` 中所有 `apiKey` 字段为空
4. 不要在公共电脑上使用，避免 API Key 泄露
5. 泄露 API Key 可能导致**经济损失**，请妥善保管

### 支持的 AI 服务商
| 服务商 | 配置标识 | 默认模型 |
|-------|---------|---------|
| 通义千问（阿里云百炼） | tongyi | qwen-turbo |
| OpenAI | openai | gpt-4o-mini |
| DeepSeek | deepseek | deepseek-chat |
| 硅基流动 | siliconflow | Qwen/Qwen2.5-7B-Instruct |

### 数据存储说明
所有游戏数据（宠物、积分、生命值、错题本等）均存储在浏览器 `localStorage` 中：
- **清除浏览器缓存**会丢失所有数据
- **无云同步功能**
- 数据仅保存在当前设备

## 🛠️ 技术栈

| 层级 | 技术选型 |
|------|---------|
| 前端框架 | 原生 HTML + CSS + JavaScript（无框架依赖） |
| 数据存储 | localStorage |
| AI 接口 | OpenAI 兼容 API（支持通义千问/DeepSeek/硅基流动等） |
| 本地 NLP 评测 | 同义词匹配 + Jaccard 相似度 + Levenshtein 编辑距离 |
| 数据存储 | 本地缓存（wx.getStorageSync） |
| 题库 | 本地JSON（500+词汇，三级分类） |
| AI接口 | DeepSeek API（可选增强功能） |

## 🚀 快速开始

### 前提条件
1. 下载安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 注册微信小程序账号（可选，无AppID也可预览调试）

### 运行步骤
1. 打开微信开发者工具
2. 选择「导入项目」
3. 项目目录选择 `miniprogram/`
4. 选择「使用测试号」（无需AppID）
5. 点击「编译」即可预览

## 📊 游戏数据存储

所有数据存储在微信小程序的本地缓存中：
- 用户积分、生命值
- 选中的宠物信息
- 背包物品（食品/玩具）
- 答题历史

清除缓存或卸载小程序会丢失数据。
